# RD Credibility Dashboard

A comprehensive tool for estimating and visualizing research and development credibility metrics.

[![PyPI Version](https://img.shields.io/pypi/v/rd-credibility)](https://pypi.org/project/rd-credibility/)
[![GitHub Actions](https://img.shields.io/github/actions/workflow/status/your-repo/rd-credibility/ci.yml)](https://github.com/your-repo/rd-credibility/actions)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## Quickstart

Install the package:

```bash
pip install rd-credibility
```

Run the dashboard:

```bash
streamlit run -m rd_credibility.app
```

Open your browser to `http://localhost:8501` to get started.
